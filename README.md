# CryptoHunk

Developed an Cryptocurrency Tracker App using ReactJS, data is pulled using an API which gets updated in every 5 minutes. We can search here for any crypto currency to find out their latest prices, market cap, volume and percentage of price increased or decreased today
## API Reference


https://api.coingecko.com/api/v3/coins/markets?vs_currency=inr&order=market_cap_desc&per_page=250&page=1&sparkline=false
